
# Cybersecurity in Banking and Cloud Computing (Java + Spring Boot)

This project is a **demo banking system** showing common threats and defenses using Spring Boot as backend and a static frontend.

**Includes**
- Java Spring Boot backend (`backend/`)
- Frontend (`frontend/`)
- Docs (`docs/`)
- Docker setup (`docker/`)

Run backend:
```bash
cd backend
./mvnw spring-boot:run
```

Open `frontend/index.html` in browser (API base = http://localhost:8080/api).
